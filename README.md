## 프로젝트 개요
- 정보 업데이트 배치처리
- 배치처리 스케쥴링
- 투자화면/투자상세화면/보유종목 화면 조회
- 투자금 입금 구현
  - Phase 1 : 입금 거래 정보 서버 등록
  - Phase 2 : Phase 1 등록 거래 정보 검증 후 실제 고객 자산 업데이트
